# 🎮 Fuga da Masmorra (Dungeon Escape)

Jogo simples no estilo roguelike desenvolvido com Python e PgZero.

---

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como parte de um teste para tutor de Python.

O objetivo do jogo é simples:
- Controlar o personagem
- Evitar os inimigos
- Sobreviver por 20 segundos para vencer

---

## 🕹️ Como Jogar

- Use as setas do teclado (⬆️⬇️⬅️➡️) para mover o personagem
- Evite os inimigos que se movem pelo mapa
- Se encostar em um inimigo, o jogo reinicia
- Sobreviva por 20 segundos para ganhar

---

## 🧩 Funcionalidades

- Menu principal com botões:
  - Iniciar jogo
  - Ligar/Desligar som
  - Sair
- Sons e música de fundo
- Animação de sprites (personagem e inimigos)
- Inimigos com movimento automático
- Sistema de tempo e condição de vitória
- Tela de vitória

---

## 🛠️ Tecnologias Utilizadas

- Python
- PgZero
